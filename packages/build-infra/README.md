# Build Infrastructure

Shared build utilities for Socket CLI.

## Exports

- **esbuild-plugin-unicode-transform** - Unicode property escape transforms for --with-intl=none
- **extraction-cache** - Hash-based caching to skip unchanged extractions
- **script-runner** - Monorepo script orchestration utilities
- **setup-helpers** - Prerequisite checking and auto-install for development setup
- **unicode-property-escape-transform** - Low-level Unicode regex transformations
